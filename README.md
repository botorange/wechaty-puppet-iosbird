# SETTING

1. Clone
```
git clone https://github.com/Chatie/wechaty.git
git clone https://github.com/Chatie/wechaty-puppet.git
```

2. Change dir

- change all `wechaty` related to `../wechaty/src`
- change all `wechaty-puppet` related to `../../wechaty-puppet/src`

3. Run

```
npm run start
```
